# LUCKY-MD-DEPLOY

---

***Deploy on [`VERCEL`](https://vercel.com/login).***

```
VERCEL PROCESS:
    1: Click "NEW".
    2: Select "New Project".
    3: Click "Search and import from your forked Git repository".
    4: Enter project name, env variables.
    5: And JUST CLICK "Deploy". 
```
